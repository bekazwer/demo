============================================================
  LOVE QUEST — Инструкция по настройке
============================================================

Все настройки находятся в одном файле: data.js
Откройте его в любом текстовом редакторе (Блокнот, VS Code).


------------------------------------------------------------
  КАК ЗАМЕНИТЬ ИМЕНА
------------------------------------------------------------ 
Найдите строку:
  names: { from: "Алишер", to: "Айгерим" },

  from — ваше имя (чьё это признание)
  to   — имя любимого человека

Замените "Алишер" и "Айгерим" на нужные имена.


------------------------------------------------------------
  КАК ДОБАВИТЬ ФОТО (для игры ПАЗЛ)
------------------------------------------------------------
1. Положите файл фотографии в папку images/
   Пример: images/myphoto.jpg

2. В файле data.js найдите строку:
   photos: ["images/1.jpg"],

3. Замените "images/1.jpg" на путь к вашему фото:
   photos: ["images/myphoto.jpg"],

Фото лучше использовать квадратное (одинаковая ширина и высота).


------------------------------------------------------------
  КАК ИЗМЕНИТЬ СЕКРЕТНОЕ СЛОВО (игра СЛОВО)
------------------------------------------------------------
Найдите строку:
  secretWord: "АЛМАТЫ",

Замените "АЛМАТЫ" на любое слово заглавными буквами.
Например: secretWord: "МОСКВА",

Также измените подсказку:
  secretWordHint: "город в котором мы встретились",


------------------------------------------------------------
  КАК ИЗМЕНИТЬ ВАРИАНТЫ ПЕСНИ (игра ПЕСНЯ)
------------------------------------------------------------
1. Положите аудиофайл в папку audio/
   Пример: audio/mysong.mp3

2. Укажите путь к файлу:
   songFile: "audio/mysong.mp3",

3. Напишите три варианта ответа:
   songOptions: ["Первый вариант", "Второй вариант", "Третий вариант"],

4. Укажите номер правильного ответа (считаем с 0):
   songAnswer: 1,   ← это второй вариант в списке
   songAnswer: 0,   ← это первый вариант
   songAnswer: 2,   ← это третий вариант


------------------------------------------------------------
  КАК ДОБАВИТЬ ФИНАЛЬНОЕ ВИДЕО
------------------------------------------------------------
1. Положите видеофайл в папку video/
   Поддерживаемые форматы: mp4, webm

2. В файле data.js найдите строку:
   chestVideoUrl: "",

3. Укажите путь к видео:
   chestVideoUrl: "video/mysurprise.mp4",

Если видео не нужно — оставьте пустую строку: chestVideoUrl: ""


------------------------------------------------------------
  КАК ИЗМЕНИТЬ ФИНАЛЬНОЕ ПОСЛАНИЕ
------------------------------------------------------------
Найдите строку:
  chestMessage: "Текст послания...",

Замените текст на своё послание.
Текст может быть любой длины — экран прокручивается.


------------------------------------------------------------
  КАК СБРОСИТЬ ПРОГРЕСС
------------------------------------------------------------
Способ 1 (прямо на сайте):
  Внизу главного экрана есть кнопка «СБРОСИТЬ ПРОГРЕСС».
  Нажмите её — все ключи сбросятся и страница перезагрузится.

Способ 2 (через браузер):
  Откройте DevTools (F12) → вкладка Application → Local Storage
  Удалите ключ loveQuestKeys


------------------------------------------------------------
  СТРУКТУРА ПАПОК
------------------------------------------------------------
  index.html     — главный файл сайта (не редактировать)
  data.js        — все настройки (редактировать здесь)
  audio/         — аудиофайлы для игры ПЕСНЯ
  images/        — фото для игры ПАЗЛ
  video/         — видео для финального экрана


------------------------------------------------------------
  КАК ОТКРЫТЬ САЙТ
------------------------------------------------------------
Просто откройте файл index.html в браузере Chrome или Firefox.
Для лучшего опыта используйте режим полного экрана (F11).

Если файлы audio/video не воспроизводятся локально —
запустите через простой веб-сервер:
  npx serve .
  или установите расширение «Live Server» в VS Code.
